Analysis of MYIMAGE
===

**Parameters**
--

- Date: **MYDATE**
- End: **MYSTOP**
- Operating system: **MYOS**
- Java Version: **MYJAVA**
- ImageJ Version: **MYIJ**
- Image resolution: **MYRESOLUTION**
- Number of slices: **MYSLICES**

**Output**
--

- Number of identified droplets: **MYDROPLETS**
- Number of droplets in Neuropil: **MYNEUROPIL**
